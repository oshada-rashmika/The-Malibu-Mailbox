# 💌 The Malibu Mailbox
### *A love story, delivered daily…*

> “Some love letters aren’t written all at once… they arrive, one magical day at a time.” ✨

---

## 🌹 What is this?

**The Malibu Mailbox** is a soft, romantic, and slightly addictive web experience where love unfolds daily.

It’s a private digital sanctuary where someone special logs in to:
- 💖 Receive **daily love letters**
- ⏳ Watch the **countdown to the next surprise**
- 🎟️ Redeem adorable **date vouchers**
- 📜 Revisit cherished memories in a **letter archive**

And behind the scenes…  
You’re the secret author, quietly crafting moments that feel like magic.

---

## 💫 Core Experience

### 💌 Daily Letters
Every day unlocks a new heartfelt message — like opening a real envelope, but softer, warmer, and glowing with intention.

### ⏳ Midnight Countdown
A delicate animated timer builds anticipation…  
Because love feels even sweeter when you wait for it.

### 🎟️ Digital Vouchers
A playful wallet of redeemable moments:
- “Good for one iced coffee ☕”
- “Movie night, your choice 🎬”
- “One long hug, no expiry 🤗”

Tap. Redeem. Celebrate. 🎉

### 📜 Memory Archive
Every opened letter becomes part of a growing love story — preserved forever.

## 🧠 Architecture (a.k.a. how the magic works)

This project blends modern web tech with soft romantic energy:

### 🌐 Frontend
- **Next.js (App Router + TypeScript)**
- **Tailwind CSS** with a *luxurious romantic palette*:
  - 🌸 Blush Pink
  - 🌹 Rose Gold
  - 🤍 Silk White
  - 🍷 Deep Velvet
- **Framer Motion** for dreamy animations
- Glassmorphism UI for that soft, glowing feel

### 🔐 Backend
- **Express.js + TypeScript**
- Clean REST API structure:
  - `/api/letters`
  - `/api/vouchers`

### 🗄️ Database
- **Supabase (PostgreSQL)**
- Secure **Row Level Security (RLS)**
- Authenticated, private, and safe
